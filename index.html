Flashcards

  • JavaScript (JS)

    • An object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • Variables

    • a data item that may take on more than one value during the runtime of a program

  • Functions

    • a set of statements that performs a task or calculates a value.

  • Methods

    • the actions that can be performed on objects

  • Arguments

    • local variable available within all functions

  • Parameters

    • names listed in the funtion

  • Return

    • stops the execution of a function and returns a value from that function.

  • Chaining methods

    • also known as Cascading, refers to repeatedly calling one method after another on an object,

  • Strings

    • is any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks).

  • Booleans

    • true or false

  • Undefined

    • undefined are both are used to represent the absence of some value.

  • NaN

    • Stands for not a number, though considered a number

  • Escape

    • function encodes a string. This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

  • Alert

    • displays an alert box with a specified message and an OK button

  • Comments

    • JavaScript comments can be used to explain JavaScript code, and to make it more readable.
